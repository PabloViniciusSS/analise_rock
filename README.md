
# Análise de Amostras de Rochas Lunares

Projeto de análise dados, sobre amostras de rochas lunares em missões espaciais Apollo e a previsão da quantidade de amostras de rochas que as novas missões lunares vão poder trazer a Terra, comparando os pesos dos módulos e as quantidades que foram trazidas anteriormente, pode ser feito uma média para prever a quantidade que poderá ser trago, além de identificar a quantidade de cada tipo de rocha.

# Projeto

Para ver o projeto é so clicar no arquivo .ipynb.
Se quiser ver localmente, é preciso clonar o projeto e ter na sua máquina o python instalado e alguma ferramenta que possa abrir o caderno, como por exemplo o Jupyter notebook.


    
## Aprendizados

O objetivo é conhecer mais sobre análise de dados, aprender as funcionalidade do uso do python com a utilização da biblioteca pandas, o projeto foi desenvolvido com base no projeto que esta na Microsoft.
